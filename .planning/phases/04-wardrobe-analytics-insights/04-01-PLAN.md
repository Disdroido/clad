Create `server/utils/gap-analysis.ts` with the gap detection rules engine:

1. **Define `InventorySummary` interface** as shown in `<interfaces>` above.

2. **Define `DetectedGap` interface** as shown in `<interfaces>` above.

3. **Implement `runGapRules(summary: InventorySummary): DetectedGap[]`** with these rules:

   - **Missing categories:** Check every clothing type in the enum (t-shirt, shirt, blouse, sweater, hoodie, jacket, coat, jeans, trousers, shorts, skirt, dress, shoes, accessory, other). If `byType[type]` is 0 or undefined, push a `{ type: 'missing_category', detail: type, severity: 'high' }` gap. Skip 'other' and 'accessory' from mandatory detection (they're not essential categories).

   - **Tops-to-bottoms ratio:** Define TOPS = new Set(['t-shirt', 'shirt', 'blouse', 'sweater', 'hoodie']), BOTTOMS = new Set(['jeans', 'trousers', 'shorts', 'skirt']). Sum counts from byType. If topsCount > 0 and bottomsCount === 0, push gap `{ type: 'missing_category', detail: 'bottoms', severity: 'high' }`. If both > 0 and tops/bottoms ratio > 3, push imbalance gap. If bottomsCount > 0 and topsCount === 0, push 'tops' missing_category gap.

   - **Season coverage:** If `bySeason['winter']` is 0 or undefined (and totalItems > 5), push `{ type: 'season_gap', detail: 'winter', severity: 'medium' }`. Same for 'summer'.

   - **Colour concentration:** If totalItems > 3, check if any single colour is >= 40% of totalItems. If so, push `{ type: 'colour_concentration', detail: colourName, severity: 'low' }`.

4. **Export all interfaces and `runGapRules`** as named exports.

Use the existing codebase conventions: no default exports, descriptive variable names.

Create `server/api/analytics/index.get.ts` returning all analytics data in one call:

1. **Auth:** Call `requireUserId(event)` at the top.

2. **Build base filter:** `and(eq(wardrobeItems.userId, userId), eq(wardrobeItems.isArchived, false))` for non-archived items.

3. **Composition query** (ANALYTICS-01):
   ```typescript
   const byType = await db
     .select({ label: wardrobeItems.clothingType, value: count() })
     .from(wardrobeItems)
     .where(baseWhere)
     .groupBy(wardrobeItems.clothingType)
     .orderBy(desc(count()))
   ```

4. **Colour palette query** (D-03, ANALYTICS-01):
   ```typescript
   const colourPalette = await db
     .select({ colour: wardrobeItems.colour, count: count() })
     .from(wardrobeItems)
     .where(baseWhere)
     .groupBy(wardrobeItems.colour)
     .orderBy(desc(count()))
   ```

5. **Season breakdown** (ANALYTICS-01):
   ```typescript
   const bySeason = await db
     .select({ season: wardrobeItems.season, count: count() })
     .from(wardrobeItems)
     .where(baseWhere)
     .groupBy(wardrobeItems.season)
     .orderBy(desc(count()))
   ```

6. **Most-worn items** (D-05, ANALYTICS-02):
   Use application-level aggregation for reliability (avoids raw SQL type casting issues with Neon HTTP driver):
   
   a. Fetch all wear events for this user with outfit itemIds:
   ```typescript
   import { innerJoin } from 'drizzle-orm'
   
   const wearEvents = await db
     .select({ outfitId: outfitWearEvents.outfitId, itemIds: outfits.itemIds })
     .from(outfitWearEvents)
     .innerJoin(outfits, eq(outfitWearEvents.outfitId, outfits.id))
     .where(eq(outfitWearEvents.userId, userId))
   ```
   
   b. Count per item ID using a Map:
   ```typescript
   const wearCounts = new Map<string, number>()
   for (const event of wearEvents) {
     for (const itemId of event.itemIds ?? []) {
       wearCounts.set(itemId, (wearCounts.get(itemId) ?? 0) + 1)
     }
   }
   ```
   
   c. Sort and take top 10:
   ```typescript
   const topEntries = [...wearCounts.entries()]
     .sort((a, b) => b[1] - a[1])
     .slice(0, 10)
     .map(([itemId, wearCount]) => ({ itemId, wearCount }))
   ```
   
   d. Fetch full item details:
   ```typescript
   import { inArray } from 'drizzle-orm'
   
   const itemIds = topEntries.map(e => e.itemId)
   const items = itemIds.length > 0
     ? await db.select().from(wardrobeItems).where(inArray(wardrobeItems.id, itemIds))
     : []
   const itemsMap = Object.fromEntries(items.map(i => [i.id, i]))
   
   const mostWorn = topEntries
     .map(e => ({ ...itemsMap[e.itemId], wearCount: e.wearCount }))
     .filter(r => r.id) // only include items that still exist
   ```

7. **Return shape:**
   ```typescript
   return { composition: byType, colourPalette, mostWorn, seasonBreakdown }
   ```

**Important:** Import all needed Drizzle helpers (`count`, `desc`, `eq`, `and`, `inArray`, `innerJoin`) from `drizzle-orm`. Import `useDb` from `~~/server/db`. The response automatically serializes to JSON via Nuxt's `defineEventHandler`.
