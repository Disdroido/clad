**Temperature scoring logic** (added to the scoring loop):
```typescript
// Temperature scoring bonus (D-03)
if (profile.currentTemperature !== undefined) {
  const temp = profile.currentTemperature
  for (const item of base) {
    if (temp < 50) {
      // Cold: boost coats, jackets, sweaters, hoodies; penalize shorts, t-shirts
      if (['coat', 'jacket', 'sweater', 'hoodie'].includes(item.clothingType)) score += 3
      if (['shorts', 't-shirt'].includes(item.clothingType)) score -= 3
    } else if (temp > 70) {
      // Warm: boost light fabrics; penalize heavy layers
      if (['t-shirt', 'shorts', 'skirt'].includes(item.clothingType)) score += 3
      if (['coat', 'sweater', 'hoodie'].includes(item.clothingType)) score -= 3
    } else {
      // Mild (50-70): small flexibility bonus
      score += 1
    }
  }
}
```

This same logic also applies to outerwear items (at the outerwear evaluation points in the loop).
