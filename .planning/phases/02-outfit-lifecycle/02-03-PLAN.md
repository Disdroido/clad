<objective>
Make outfit generation wear-aware by deprioritizing recently worn items.

**Purpose:** Users shouldn't get outfit suggestions that include items they just wore yesterday. By factoring wear recency into the outfit scoring, the AI naturally avoids suggesting recently worn combinations.

**Output:**
- Updated outfit engine scoring algorithm that considers wear recency
- Updated generate endpoint that passes wear data to the engine
</objective>

**Existing outfit engine** at `server/utils/outfit-engine.ts`:

```typescript
export interface Item {
  id: string
  clothingType: string
  colour: string
  pattern: string
  material: string
  formalityLevel: string
  season: string
  imageUrl: string
}

export interface UserProfile {
  preferredColours?: string[]
  dislikedColours?: string[]
  formalityDefault?: string
  stylePreferences?: string[]
  climate?: string
}

export function generateValidOutfits(
  items: Item[],
  profile: UserProfile,
  occasion: string
)
```

The function:
1. Filters items by season, formality, disliked colours
2. Groups by category (tops, bottoms, outerwear, shoes, full-body)
3. Generates combinations (top+bottom or dress + optional outerwear + shoes)
4. Checks colour harmony and pattern conflicts
5. Scores combinations (preferred colours +2, layering +1, shoes +1)
6. Returns top 20 combinations sorted by score descending

**Existing generate endpoint** at `server/api/outfits/generate.post.ts`:

The endpoint:
1. Fetches all wardrobe items for user
2. Fetches user profile
3. Calls `generateValidOutfits(items, profile, occasion)`
4. Sends top candidates to AI for reasoning
5. Returns best outfit with explanation

**Wear data availability** (from Plan 02-01):

The `outfits` table now has:
- `wearCount: integer` — total times worn
- `lastWornAt: timestamp` — most recent wear date

The `outfitWearEvents` table has per-wear timestamps that can be joined to find which items were recently worn.

Since outfits are composed of items (via `itemIds`), we need to determine which items were "recently worn" by looking at recent wear events and their associated outfit item IDs.
